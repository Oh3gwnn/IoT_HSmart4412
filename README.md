>
>   문양 해석, 미로 찾기, 암호 풀이, 선 끊기 4개의 게임으로 구성
>
>   전체적인 게임 풀이 틀은 같지만, 랜덤 시드 값으로 내용은 랜덤으로 구성
>
>   자세한 내용은 제작한 문서 참고
