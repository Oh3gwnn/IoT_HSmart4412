### **3. 피드백**

 - 문서 복원이나 그런 기법에 대한 논문 등을 잘 확인하고 학습해야 함.
 - **시험지 데이터 수집**: 1,000장 이상의 데이터 셋을 구해야 함.
인터넷에서도 찾고 필요하다면 졸업작품 재료비로 문제집 구매.
 - **시험지 데이터 증식**: 시험지에 낙서를 만드는 것을 자동화하여 
데이터를 많이 늘릴 수 있으면 좋을 것
